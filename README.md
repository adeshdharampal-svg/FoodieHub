# FoodieHub

FoodieHub is a modern and responsive food ordering website created using HTML, CSS, and JavaScript.  
The website includes delicious food items like pizza, burgers, fries, cakes, and rolls with an attractive user interface and smooth design.

## Features
- Responsive design
- Food menu section
- Modern homepage
- Navigation bar
- Interactive buttons
- Stylish food cards

## Technologies Used
- HTML
- CSS
- JavaScript
